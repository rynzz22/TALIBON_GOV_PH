# TALIBON.GOV.PH Documentation

This folder contains the project documentation package for MVP delivery and deployment readiness.

## Documents

- `MVP_OVERVIEW.md`  
  Scope, goals, non-goals, and architecture context for the MVP.

- `FLOWS.md`  
  User and system flows (content viewing, auth callback, admin edits, payment flow, and deployment flow).

- `DEVELOPER_GUIDE.md`  
  Local setup, environment variables, project structure, commands, testing, and release checklist.

- `MILESTONE_SPRINTS.md`  
  Suggested sprint plan with priorities, deliverables, and exit criteria.

## Suggested Reading Order

1. `MVP_OVERVIEW.md`
2. `FLOWS.md`
3. `DEVELOPER_GUIDE.md`
4. `MILESTONE_SPRINTS.md`

## Diagrams

`MVP_OVERVIEW.md` and `FLOWS.md` include [Mermaid](https://mermaid.js.org/) diagrams (` ```mermaid ` blocks inside `<details>` for compact reading). GitHub renders Mermaid automatically in Markdown files.
